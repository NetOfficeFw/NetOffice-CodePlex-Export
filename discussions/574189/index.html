I have a new problem, which is with excel, not with the NET.OFFICE, but its related and i hoped you will be able to give me a clue.

When my addon opens it tries to do the following code line, which fails:
Object ExcelProxy = System.Runtime.InteropServices.Marshal.GetActiveObject("Excel.Application");
