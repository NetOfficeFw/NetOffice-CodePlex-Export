The Developer Toolbox 

The NetOffice Developer Toolbox is a Productivity Tool for .NET developers who create solutions with and for Microsoft Office. It includes the following functions

Feature Table:

1.

Office Compatibility

Check your .NET assemblies for compatibility against several MS-Office versions

2.

Application Observer

Monitor running MS-Office applications

3.

Registry Editor

Clone of the Windows-registry editor for the MS-Office Hive Keys

4.

Addin Guard

Monitor the LoadBehaviour of COMAddins

5.

Outlook Security

Supress the Outlook Security Dialog

6.

Office UI

Find and delete dead UI Items

7.

Project Wizard

Project assistant for VS(Express Solutions)

Click on an entry in the table for further information.
