Was tut der Projekt Wizward ür Sie?

NetOffice bietet für das erstellen neuer Projekte in Visual Studio den NetOffice Projekt Wizard an. Dieser wird jedoch von Visual Studio Express nicht unterstützt da diese Version von Visual Studio keine Erweiterungen lädt. Als hilfreiche Ergänzung bietet die Developer Toolbox den Projekt Wizard als eigenständige Anwendung an. Erstellte Projekte werden im Visual Studio Express Format erzeugt.


Screenshot




Vorgehensweise

Klicken Sie im linken oberen Bereich auf die Schaltfäche "Neues Projekt" und wählen Sie im folgenden Dialog Ihre gewünschte Programmiersprache und Visual Studio Express Version aus.


Screenshot: Auswahldialog

 

Folgen Sie danach dem Assistenten. Der Projekt Wizard wird automatisch den erstellen Solution Ordner im Windows Explorer für Sie öffnen. Klicken Sie auf die generierte Solution um Sie mit Ihrer Visual Studio Express Installation zu öffnen.
