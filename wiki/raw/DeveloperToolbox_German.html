Die Developer Toolbox 

Die NetOffice Developer Toolbox ist ein Productivity Tool für .NET Entwickler die Lösungen in und für Microsoft Office enwickeln. Es umfasst die folgenden Funktionalitäten:

Feature Table:

1.

Office Compatibility

Überprüft Ihre erstellten .NET Assemblies auf MS-Office Versionskompatibilität

2.

Application Observer

Überwacht laufende MS-Office Anwendungen

3.

Registry Editor

Klone des Windows-Registry Editors für MS-Office Hive Keys

4.

Addin Guard

Überwacht das Ladeverhalten von COMAddins

5.

Outlook Security

Unterdrückt den Outlook Security Dialog

6.

Office UI

Findet und entfernt ungewollte UI Elemente

7.

Project Wizard

Projekt Assistent für Visual Studio(Express) Lösungen

 

Klicken Sie auf einen Eintrag in der Tabelle für weitere Informationen.
