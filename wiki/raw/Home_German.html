NetOffice bietet Lösungen für Office Entwickler in .NET Sprache auswählen: english | deutsch

.NET Wrapper Assemblies für den Zugriff auf folgende MS Office Anwendungen:

  • Office, Excel, Word, Outlook, PowerPoint, Access, Project, Visio

Features

  • Office Integration ohne Versionsbeschränkung
  • Alle Funktionalitäten der Versionen 2000, 2002, 2003, 2007, 2010, 2013 enthalten
  • Aktive Unterstützung bei der versionsunabhängigen Entwicklung (siehe unten)
  • Syntaktisch und semantisch identisch zu den Microsoft Interop Assemblies
  • Keine Einarbeitung wenn Sie das Office Objekt Modell bereits kennen
  • Ihr bestehender PIA Code funktioniert auch mit NetOffice
  • Reduzierter Code durch automatische Verwaltung von COM Proxies
  • Mit jeder .NET Version ab 2.0 verwendbar
  • Problemlose Addin Entwicklung
  • Keine Deployment Hürden, Registrierung
  • Keine Abhängigkeiten, Interop Assemblies oder VSTO notwendig
  • Projektvorlagen und Assistenten für Visual Studio verfügbar Fotostrecke
