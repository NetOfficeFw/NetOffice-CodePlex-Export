hello

its necessary to tell SharpDevelop the assembly location. extract the .Dll's in any location you want. a good place is the %personal userfolder%\NetOffice1.5.1

- create a new sharp develop project
- go to the reference(s) node -> right mouse click / add reference and follow the dialog
the next step you choose the assembly location, in our example %personal userfolder%\NetOffice1.5.1\%NETVersion%\Assemblies
and select the target NetOffice assemblies for your target office application. dont forget the using (Import in VB) statements as top of your code own code files.

which assembly you need for which office application
is defined in a table here: http://netoffice.codeplex.com/documentation
