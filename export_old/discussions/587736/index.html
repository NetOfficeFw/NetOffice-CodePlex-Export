Try one of the examples in release folder.

Just like:

Excel.Application app = new Excel.Application();
var book = app.Workbooks.Add();
var sheet = book.Workseets[1] as Excel.WorkSheet;
var value = sheet.Cells["A5"].Value; // get value from A5
